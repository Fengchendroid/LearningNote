git代码指令
git init                 创建本地仓库
git clone                将远程仓库与本地仓库创建连接(clone后面加远程仓库地址)
git add .                添加相关文件（可具体添加，也可以全部添加）
git commit -m ""         添加每次提交修改的注释信息
git push origin master   提交修改
git status               查看文件修改情况
git diff                 看到更改内容
git checkout             撤销更改
git reset                取消添加
